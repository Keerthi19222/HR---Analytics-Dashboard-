# HR---Analytics-Dashboard-
HR Analytics Dashboard using Power BI

📊 HR Analytics Dashboard
📌 Project Overview

This project focuses on analyzing employee data to understand workforce trends and reduce attrition. The HR Analytics Dashboard provides clear insights into employee behavior, helping organizations make better HR decisions.

🎯 Objectives
Analyze employee attrition patterns
Identify key factors affecting employee turnover
Improve employee retention strategies
Provide data-driven HR insights
🛠️ Tools & Technologies
Power BI (Dashboard & Visualization)

The dataset includes:
Atrrtion by age group
Attrtion by gender 
Attrtion by department 
Employee performance
Avg monthly income by job role
salary hike percentage by department

📈 Key Insights
Employees aged 26–35 have the highest attrition.
Male employees (~60%) contribute more to attrition than females (~40%).
Research & Development has the highest attrition.
Managers & Research roles have highest salaries.
